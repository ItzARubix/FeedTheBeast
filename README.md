This is a Murder Drones fangame, written in Java, using LWJGL and no game engine.

In order to run the project, clone the repository, cd into it, and do `./run.sh HelloWorld.java`.

Right now it is only confirmed to be working for Linux (it runs under Xwayland because the GLFW version that is provided with LWJGL is not very Wayland friendly yet) but the intention is for it to work for all operating systems.

There's very little here right now.
